# Notes App

A full-stack Notes application built to practice and strengthen my understanding of React, Express, and MongoDB. The project focuses on implementing core CRUD functionality, authentication, and backend integration while following modern web development practices.

## Features

- Create, read, update, and delete notes
- User authentication and authorization
- Secure backend API with Express
- MongoDB database integration
- Responsive and interactive React frontend

## Tech Stack

### Frontend

- React

### Backend

- Node.js
- Express.js

### Database

- MongoDB
- Mongoose

## Learning Goals

This project is being built primarily as a learning exercise to gain hands-on experience with:

- React fundamentals
- REST API development
- Express middleware and routing
- MongoDB data modeling
- Authentication and authorization
- Full-stack application architecture
